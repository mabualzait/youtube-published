How to authenticate with YouTube:

  1. Head to Google Cloud Console, enable "YouTube Data API v3".
  2. Create OAuth 2.0 Credentials (Desktop App) and download the JSON.
  3. Paste the raw JSON content directly into the text area above and Save.
  4. IMPORTANT: Open the terminal on your server and manually run python auth_youtube_agent.py once to sign in. It will generate a permanent token_youtube.json authorizing this machine!
